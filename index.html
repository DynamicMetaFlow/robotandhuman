Robot & Human

About this comic

Robot & Human is a JSON-powered webcomic. What that means is that I drew some graphics and wrote some JavaScript and now all I have to do to make a new strip is write it in JSON. If you think that sounds easier than just drawing a picture and uploading it, you might be a programmer.

If you want to set up this tool on your own site, you'll need to copy (and then probably modify) these files:

  • index.html (i.e. this page)—you'll probably want to modify it heavily, but you should keep the JS/CSS calls in the head, and note that comics.js needs the element with an ID of "place" so it knows where to insert things.
  • jQuery 2.0.1, or another version if you prefer; I'm not aware of any compatibility issues
  • comics.js (the "engine")
  • rhwc.css—again, you'll probably make a lot of changes to this, but the code that comics.js produces won't make any sense at all without a lot of styling
  • maybe comics.json, just to see the format, although this is the bulk of the comic's content, so you'll eventually want to delete what's there and write your own
  • the image files human-fl.png, human-fr.png, robot-fl.png, and robot-fr.png, and/or you can make your own graphics

P.S.: If the comics don't display right (or at all), you might be using an outdated web browser. Please keep current! Web developers everywhere will thank you for it.
